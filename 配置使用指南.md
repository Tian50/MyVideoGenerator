## 配置文件结构

### APIConfig 类
管理所有API参数配置：

```python
class APIConfig:
    # DeepSeek API 配置
    DEEPSEEK_CONFIG = {
        "translation": {
            "model": "deepseek-chat",
            "temperature": 0.3,
            "max_tokens": 2000
        },
        # ... 其他任务配置
    }
    
    # 重试配置
    RETRY_CONFIG = {
        "max_retries": 3,
        "backoff_factor": 2,
        "timeout": 60
    }
```

### 提示词类
按功能模块组织的提示词：

1. **TranslatorPrompts** - 翻译相关提示词
2. **TitleGeneratorPrompts** - 标题生成提示词
3. **BloggerInfoPrompts** - 博主信息提示词
4. **HighlightDetectorPrompts** - 高光检测提示词
5. **ContentFilterPrompts** - 内容过滤提示词
6. **CoverGeneratorPrompts** - 封面生成提示词

### PromptManager 类
统一的访问接口：

```python
class PromptManager:
    @staticmethod
    def get_api_config(task_type):  # 获取API配置
    @staticmethod
    def get_retry_config():  # 获取重试配置
    @staticmethod
    def get_translation_prompt(target_lang="zh"):  # 获取翻译提示词
    # ... 其他提示词获取方法
```

## 如何使用

### 1. 查看当前配置
运行配置验证工具：
```bash
python config_validator.py
```

### 2. 修改配置
编辑 `prompts_config.py` 文件：

**修改API参数**：
```python
# 修改翻译任务的温度
APIConfig.DEEPSEEK_CONFIG["translation"]["temperature"] = 0.5

# 修改重试配置
APIConfig.RETRY_CONFIG["max_retries"] = 5
```

**修改提示词**：
```python
class TitleGeneratorPrompts:
    @staticmethod
    def get_title_generation_prompt():
        # 修改这里的提示词内容
        return '''{title_ref}{meta_desc}
下面的是视频字幕文本，请为这个视频起 **20 个**吸引人的标题...
（这里添加你的自定义要求）'''
```

### 3. 在代码中使用配置
```python
from prompts_config import PromptManager

# 获取API配置
api_config = PromptManager.get_api_config("translation")
retry_config = PromptManager.get_retry_config()

# 获取提示词
translation_prompt = PromptManager.get_translation_prompt("zh")

# 格式化提示词（带参数）
title_prompt = PromptManager.get_title_generation_prompt().format(
    title_ref="视频原标题：测试\n",
    meta_desc="视频元数据：博主：张三"
)
```

## 故障排除

### 常见问题

1. **导入错误**
   ```
   ModuleNotFoundError: No module named 'prompts_config'
   ```
   **解决方案**：确保 `prompts_config.py` 文件在项目根目录

2. **配置验证失败**
   ```
   配置验证失败，发现以下错误：
     1. API配置 'translation' 缺少 model 字段
   ```
   **解决方案**：检查 `prompts_config.py` 中的配置格式

3. **提示词调用失败**
   ```
   标题生成提示词: 调用失败 - KeyError: 'title_ref'
   ```
   **解决方案**：确保提示词中的占位符正确格式化

### 调试建议

1. 运行验证工具检查配置
2. 查看具体错误信息
3. 检查配置文件格式
4. 确保所有模块正确导入
